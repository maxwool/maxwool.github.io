About Me

Hey — I'm Max, a senior at CU Boulder studying Computer Science and Mathematics. I'm into data analytics, machine learning, and building software that actually solves problems. Whether that's a computer vision system for jigsaw puzzles or a quantitative model for sports betting, I like working where math and code intersect.

This portfolio is a living document of my work, interests, and growth as a developer and analyst. It's built for recruiters and hiring managers who want to see what I've actually built — not just a résumé. If something here catches your eye, reach out.

I'm actively looking for full-time data analytics and data engineering roles starting summer 2026. I'm especially interested in sports analytics, fintech, and anywhere that data actually drives decisions.

Projects

I'm Puzzled!
Capstone · iOS · Computer Vision
I'm Puzzled!

An iOS app built with a six-person team under GriffMonster LLC that uses real-time computer vision to help puzzlers find piece placements. The ML pipeline runs Mask2Former — a universal instance segmentation architecture — to isolate individual puzzle pieces from a camera frame, then passes each piece to a SIFT-based matching algorithm that compares feature descriptors against a reference image and returns a placement suggestion. Built with an iOS Swift frontend, FastAPI backend, and an accessibility-first design philosophy. Presented at the CU Boulder Senior Capstone Expo.

Diamond Analytics
Startup · Data Analytics · MLB
Diamond Analytics

A subscription-based MLB sports betting analytics service built entirely from scratch — including branding, quantitative edge models, a subscriber-facing dashboard, and a social media presence. Diamond Analytics uses historical game data, line movement analysis, and regression modeling to identify +EV betting opportunities in MLB markets. The project involved building out the full brand identity (dark terminal aesthetic, green/white/gold palette), an ROI calculator widget, and a content strategy targeting serious bettors who value transparency and data over gut picks.
